# Mad Bomber Vita


A PSVITA port of Bill Kendrick's [Mad Bomber](http://www.newbreedsoftware.com/madbomber/)

![Mad Bomber](https://raw.githubusercontent.com/SonicMastr/Mad-Bomber-Vita/master/screenshots/1.png)

#### Controls:
**Left** - Move Left
**Right** - Move Right
**Cross** - Confirm/Fire
**Start** - Pause
**Select** - Escape/Exit
